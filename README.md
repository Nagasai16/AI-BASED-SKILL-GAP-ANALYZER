# 🤖 AI-Based Skill Gap Analyzer

This repository contains the source code for a web application that analyzes user skills and identifies gaps using intelligent techniques. The system provides personalized recommendations to help users improve their skill set and align with industry requirements.

---

## 🎯 Project Goal

The main objective is to build and deploy an intelligent system that evaluates a user's current skills and compares them with required skills for specific roles. The system suggests missing skills, learning paths, and improvements using AI-driven logic.

---

## 🗂️ Project Modules

| # | Module                | Status | Description                                                         |
| - | --------------------- | ------ | ------------------------------------------------------------------- |
| 1 | Skill Input System    | ✅      | Allows users to input their existing skills and desired role        |
| 2 | Skill Analysis Engine | ✅      | Compares user skills with required skills using rule-based/AI logic |
| 3 | Recommendation System | ✅      | Suggests missing skills and improvement areas                       |
| 4 | Web Interface         | ✅      | Flask-based UI for interaction and results display                  |
| 5 | Data Management       | ✅      | Stores user data and skill sets using SQLite                        |

---

## 📂 File Structure

| File/Folder          | Purpose                                          |
| -------------------- | ------------------------------------------------ |
| app.py               | Main Flask application handling routes and logic |
| database.db          | Stores user data and skill mappings              |

| README.md            | Project documentation                            |

## 🚀 How to Run

**Clone the Repository**
```bash
git clone https://github.com/your-username/AI-BASED-SKILL-GAP-ANALYZER.git
cd AI-BASED-SKILL-GAP-ANALYZER
```

**Install Dependencies**
```bash
pip install -r requirements.txt
```

**Run the Application**

python app.py

**Access in Browser**
```bash
Open: http://127.0.0.1:5000
```
